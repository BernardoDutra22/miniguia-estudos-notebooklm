# miniguia-estudos-notebooklm
Miniguia de estudos sobre Python e Dados utilizando notebooklm. Este guia tem o objetivo de facilitar o estudo em análise de dados utilizando a linguagem de programação Python.
Este documento sintetiza os principais conceitos, ferramentas e filosofias da linguagem de programação Python, com base em diretrizes técnicas e pedagógicas. Python consolidou-se como a linguagem mais popular da atualidade (liderando o ranking TIOBE em 2023) devido à sua sintaxe limpa, portabilidade e natureza multiparadigma. Criada por Guido van Rossum no final dos anos 80, a linguagem prioriza a legibilidade e a simplicidade, sendo amplamente aplicada em áreas críticas como Inteligência Artificial, Ciência de Dados, Automação e Desenvolvimento Web. O ecossistema Python é sustentado por uma vasta biblioteca padrão e um sistema robusto de pacotes externos (gerenciados pelo PIP), permitindo sua extensão para nichos específicos através de bibliotecas como NumPy e Matplotlib.
